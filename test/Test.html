ShadowsocksR 一键脚本（ssr.sh）是逗比根据地站长开发并维护的一个 ShadowsocksR 脚本，脚本除了可以一键安装 SSR 外，还可以限制用户速度、限制端口设备数、一键安装加速（锐速、BBR），是一个功能比较全面的 ShadowsocksR 脚本。

2、脚本介绍

之前介绍过秋水逸冰的 ShadowsocksR 脚本：秋水逸冰 Shadowsocks 四合一脚本，shadowsocks-all.sh，与其相比，逗比根据地的 ShadowsocksR 脚本除了可以安装 SSR 服务器端之外，还有一些附带功能，包括限制速度、安装加速等，这个脚本的特性总结如下：

  • 支持 限制 用户速度
  • 支持 限制 端口设备数
  • 支持 显示 当前连接IP
  • 支持 显示 SS/SSR连接+二维码
  • 支持 切换管理 单/多端口
  • 支持 一键安装 锐速
  • 支持 一键安装 BBR
  • 支持 一键封禁 垃圾邮件(SMAP)/BT/PT

逗比根据地 ShadowsocksR 一键脚本支持 CentOS 6+，Debian 6+，Ubuntu 14+。

 

3、脚本使用

在 VPS 上依执行以下下载并运行的命令：

wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/ssr.sh && chmod +x ssr.sh && bash ssr.sh

之后就是 ssr.sh 的管理界面，关于这个 ShadowsocksR 管理界面稍作解释：

  • 第一部分是关于安装/更新/卸载 ShadowsocksR 的，而 libsodium 的作用是如果你的加密方式选择 chacha20，则需要先安装 libsodium；
  • 第二部分是限制速度与连接设备的；
  • 第三部分（选项14）则是一些其他功能，例如一键安装锐速 / BBR 加速。

输入数字1后就可以开始安装 ShadowsocksR 了，之后就是要你输入一些 ShadowsocksR 的配置，包括端口、密码、加密方式、协议等，如果你不懂的话，直接选择脚本的默认值即可（直接按回车就是默认值）。这里需要注意的是，如果你选择的加密方式是 salsa20/chacha20-* 系列加密方式，需要额外安装依赖 libsodium ，否则会无法启动 ShadowsocksR：

之后 ssr.sh 脚本会询问 是否设置 协议插件兼容原版(_compatible)？[Y/n]，直接输入 n 回车即可，一般无需兼容：

接下来的三项，是限制 ShadowsocksR 的连接速度与设备数量的，普通用户一般不需要设置，直接回车即可：

全部选择完毕后，静静的等待 ssr.sh 安装 ShadowsocksR 即可，如果中途让你安装什么依赖，直接输入 y 安装：

等大概 10-15 分钟，ShadowsocksR 服务端就安装好了，需要注意的是，因为逗比根据地已经关站了，所以此处的 SSR 二维码无法打开，但是 ShadowsocksR 账号配置信息都是可以用的，配置到客户端就行了：

 

4、安装加速

ssr.sh 这个脚本自带加速，可以一键按爪给你 BBR 和锐速加速，直接执行运行脚本的命令 bash ssr.sh 即可，之后选择 14 进入其他功能，就可以安装加速了：

 

5、脚本其他设置

ssr.sh 脚本的所有设置就是从 bash ssr.sh 开始，只要在当前目录下输入该命令就可以进入 ShadowsocksR 的管理页面，这里你可以更新 ShadowsocksR、卸载 ShadowsocksR、修改限速、修改设置限制等等，可以说逗比根据地 ShadowsocksR 一键脚本的功能还是很全面的。

 

6、安装问题

如果你在安装 SSR 服务端时，出现了 ShadowsocksR 启动失败 的问题，那么可能是你的 libsodium 没有安装，只需要执行 bash ssr.sh 先安装 libsodium，之后再进入管理界面选择10启动 ShadowsocksR 就可以了。
